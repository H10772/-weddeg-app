# بعد ما تعمل Repository على GitHub، شغل الأوامر دي:

# استبدل YOUR_USERNAME باسم المستخدم بتاعك على GitHub
# استبدل YOUR_REPO_NAME باسم الـ repository اللي عملته

git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
git branch -M main
git push -u origin main

# مثال:
# git remote add origin https://github.com/ahmed/weddeg-app.git
# git branch -M main
# git push -u origin main

# ملاحظة: هيطلب منك Username و Password
# Password = Personal Access Token (مش الباسورد العادي)
# لو مش عندك Token، اعمله من:
# GitHub Settings > Developer settings > Personal access tokens > Tokens (classic) > Generate new token
